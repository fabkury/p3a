Hi Waveshare team,

My name is Fabrício, and I'm the developer of **p3a** — an open-source animated pixel art player built entirely on your **ESP32-P4-WIFI6-Touch-LCD-4B** board.

p3a turns the board into a dedicated pixel art display that plays animated WebP, GIF, PNG, and JPEG files. It connects to Giphy for trending GIFs, to Makapix Club (a pixel art social network) for curated artworks, and can also play files from the SD card. Users interact via the touchscreen and a built-in web UI accessible from any device on the same network.

The project takes advantage of several of the board's key features:

- **720x720 IPS display** — triple-buffered rendering with VSYNC for smooth animation playback
- **GT911 touch panel** — full touch-based navigation and control
- **ESP32-C6 Wi-Fi 6 co-processor** — streams artwork from the internet, hosts a captive portal for Wi-Fi setup, and serves the web UI
- **SD card** — local artwork storage and caching with SHA256-sharded vault
- **OTA updates** — wireless firmware updates pulled from GitHub Releases, including automatic co-processor firmware updates

The project is fully open-source on GitHub: **https://github.com/fabkury/p3a**
