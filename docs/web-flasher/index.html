⚠️ Web Flasher Unavailable

Due to a limitation in Espressif's esptool-js library, browser-based flashing is not currently supported for ESP32-P4 devices.

What's the issue?

The ESP32-P4 chip requires specific flash settings (DIO mode, 80MHz, 32MB) that must be explicitly configured. The browser-based esptool-js library fails to recognize the ESP32-P4 bootloader format and cannot apply these settings correctly.

Warning: Image file at 0x2000 doesn't look like an image file, so not changing any flash settings.

How to flash your p3a

Please use the command-line tool instead. It takes about 5 minutes to set up and works reliably:

  • Works on Windows, macOS, and Linux
  • Only needed once — future updates are wireless (OTA)
  • Simple copy-paste commands
→ View Flashing Guide

Will web flashing be available later?

We've reported this issue to Espressif. Once they update esptool-js to properly support ESP32-P4, we'll enable the web flasher. In the meantime, the command-line method is straightforward and reliable.
